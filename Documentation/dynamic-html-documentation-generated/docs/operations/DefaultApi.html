Response Type

computersList

Parameters

  • Path: id Integer

    ID of the computer in OCS database

computerIdSectionsGet

URL

https://api.ocsinventory-ng.tld/ocsapi/v1/computer/{id}/{sections}

HTTP Method

GET

Response Type

computerDetail

Parameters

  • Path: id Integer

    ID of the computer in OCS database

  • Path: sections String

    Name of the requested section (i.e -> bios, software, etc..)

  • Query: where String

    Search on a specific column of the specific section.

  • Query: operator String

    Search operator (like, not like, =, !=, <, >, <=, >=). Mandatory if where parameter is set.
